# Aider 风格命令完整使用手册

Clawd Code 在 REPL 环境中集成了类似 Aider 的命令操作模式，极大提升了文件交互、Git 管理和模型调试的效率。

## 基础交互命令

| 命令 | 别名 | 功能描述 |
| :--- | :--- | :--- |
| `/add [file]` |  | 将指定文件添加到当前聊天上下文中，使其被 Agent 感知。 |
| `/drop [file]` |  | 将文件从聊天上下文中移除。 |
| `/read [file]` |  | 直接读取文件内容。 |
| `/edit [file]` |  | 启动编辑模式（支持行号定位）。 |

## 执行与测试命令

| 命令 | 别名 | 功能描述 |
| :--- | :--- | :--- |
| `/run [cmd]` | `/!` | 在后台执行指定的 Shell 命令并返回输出。 |
| `/shell` |  | 启动交互式 Shell。 |
| `/test` |  | 运行项目测试用例，验证代码更改是否通过检查。 |
| `/lint` |  | 运行代码 Linter（如 `ruff` 或 `black`），确保符合规范。 |

## Git 版本管理命令

| 命令 | 别名 | 功能描述 |
| :--- | :--- | :--- |
| `/git [args]` | `/g` | 快捷 Git 操作。 |
| `/diff` |  | 显示当前未提交的代码变更。 |
| `/commit [msg]` |  | 提交当前暂存区的变更并附带指定消息。 |
| `/undo` |  | 撤销 AI 生成的最后一次代码修改。 |

## 高级模型与上下文控制

| 命令 | 别名 | 功能描述 |
| :--- | :--- | :--- |
| `/model` |  | 查看或切换当前使用的主模型。 |
| `/editor-model` |  | 切换负责代码修改的编辑器模型。 |
| `/weak-model` |  | 切换弱模型（通常用于提交信息生成或摘要）。 |
| `/chat-mode` |  | 切换聊天模式（`ask` 问答、`code` 编码、`architect` 架构师、`context` 上下文）。 |
| `/tokens` |  | 查看当前对话上下文的 Token 消耗与剩余预算。 |
| `/map` |  | 显示或调整代码地图（Code Map）的详细程度。 |
| `/browse` |  | 抓取网页内容并转换为 Markdown 格式供 Agent 参考。 |
| `/thinking` |  | 开启或关闭模型的思考过程展示。 |
